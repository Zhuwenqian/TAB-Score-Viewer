# 功能更新记录

## v2.0.7 (2026-06-15) - 异步导出+JPG格式支持

### 新功能

- **异步导出系统 (解决大文件UI冻结)**:
  - 导出操作改为后台线程(QRunnable + QThreadPool)执行，UI不再冻结
  - 新增 `ExportProgressDialog` 进度对话框:
    - 实时显示渲染状态(收集数据 / 渲染轨道X/Y / 保存文件 / 完成)
    - 进度条可视化展示整体进度
    - 取消按钮可随时中断导出
    - 完成/出错/取消后自动关闭并提示结果
  - 新增 `ExportWorker` 工作线程类: 通过信号报告进度/完成/错误/取消

- **JPG格式导出支持**:
  - ExportDialog格式选项新增 **JPG (有损压缩,适合分享)**
  - 新增JPG质量滑块(1-100，默认90)，仅选择JPG时显示
  - 质量参数说明: 90=高质量接近无损 / 80=网络分享 / 60=极致压缩
  - 文件体积比PNG小3-5倍，适合微信/QQ/论坛分享
  - 新增 `_render_to_a4_jpg()` 方法: 基于PNG渲染逻辑，使用Qt的JPEG质量参数输出

### 架构变更

- **导出流程从同步改为异步**:
  - 旧版(v2): `_export_to_a4()` → UI主线程直接调用渲染 → 大文件冻结5-10秒
  - 新版(v3): `_export_to_a4()` → 创建ExportWorker → QThreadPool后台运行 → 信号驱动进度更新
  - ExportWorker持有DisplayWindow引用，调用其渲染方法完成实际工作
  - ExportProgressDialog通过信号槽机制接收进度更新(跨线程安全)

### 技术实现

- **ExportWorkerSignals**: 自定义信号类(progress/finished/error/cancelled)，继承QObject支持pyqtSignal
- **ExportWorker(QRunnable)**: 
  - run()方法中依次执行: 收集数据 → 逐轨渲染(每轨前检查取消标志) → 报告完成
  - cancel()方法设置原子标志位，下个渲染循环生效
- **ExportProgressDialog(QDialog)**:
  - set_worker()绑定ExportWorker实例和信号连接
  - 进度条计算: 每轨道占(100/总轨道)分，当前轨道内按页码细分
  - 自动关闭: 完成500ms后 / 出错500ms后 / 取消800ms后
- **_render_to_a4_jpg()**: 
  - 复用PNG的预渲染防溢出方案(page_buffer → A4画布)
  - QImage格式用RGB32(无alpha通道，比ARGB32更省空间)
  - Qt质量映射: 用户1-100 → Qt内部0-99

### UI参数(可调整)

| 参数 | 默认值 | 说明 | 调整效果 |
|------|--------|------|----------|
| jpg_quality | 90 | JPG压缩质量(1-100) | 越大画质越好文件越大 |
| progress_dialog_width | 420px | 进度对话框宽度 | 影响对话框大小 |
| progress_dialog_height | 160px | 进度对话框高度 | 影响对话框大小 |

### 10条Case预期结果

| Case | 场景 | 预期结果 |
| --- | --- | --- |
| 1 | 单张图片谱面+选PNG格式 | 弹出进度框→快速完成→弹出成功提示→生成PNG文件 |

| 10 | 无内容时点击导出按钮 | 直接提示"无法导出"(与之前一致，不弹进度框) |

## v0.2.9 (2026-06-18) - ApolloTab渲染器标题居中+行距优化

### 修改内容

- **标题居中显示**: 页首歌曲标题从左对齐改为居中对齐(`Qt.AlignCenter`)，视觉上更加美观专业
- **行距增大**: 标题行与信息行(轨道名/调弦/BPM)之间的间距从 26px 增加到 35px，增加呼吸空间

### 修改文件

| 文件 | 改动 |
| --- | --- |
| `ApolloTab/renderer/tab_renderer.py` | `_draw_header()`: 标题绘制从`Qt.AlignLeft`改为`Qt.AlignCenter`; 行高从25px增加到30px; 行间距从26px增加到35px |

### 10条Case预期结果

| Case | 场景 | 预期结果 |
| --- | --- | --- |
| 1 | 短标题(如"夜曲") | 标题在页首水平居中显示 |

| 3 | 空标题(song.title=None) | 显示"Untitled"并居中 |
