# 功能更新记录

## v1.0.0 (2026-06-06) - 初始版本

### 新增功能

#### 核心功能
- **多格式文件支持**: PNG、JPG、JPEG、WEBP 图片格式；PDF 文档渲染；GP3/GP4/GP5/GPX 吉他谱文件预览
- **自动滚动播放**: 谱面自动向上滚动模拟演奏，支持播放/暂停控制
- **速度控制系统**:
  - 基础速度调节（1-500ms），值越小越快
  - 速度曲线编辑器：贝塞尔曲线可视化编辑，支持自定义控制点
  - 预设曲线模板：匀速、渐快、渐慢、先慢后快、先快后慢
- **循环播放功能**:
  - 不循环模式
  - 全局循环模式（从头开始）
  - A-B 区域循环模式（通过进度条 Ctrl+Shift 点击设置）
- **进度条系统**:
  - 自定义 ProgressBarSlider 组件，支持拖动跳转
  - 显示当前位置百分比和已播放时间
  - 可视化标记 A-B 循环区域

#### 标注系统
- **文本标注**: 双击谱面任意位置添加演奏技巧说明文本
- **标注属性**: 支持设置颜色、字体大小、字体族、粗体、背景色
- **标注管理器**: 对话框式批量管理所有标注
  - 新增 / 编辑 / 删除 / 清空操作
  - Ctrl+Z 撤销 / Ctrl+Y 重做（最多50步历史）
  - 取消操作自动恢复状态
- **双击编辑**: 双击已有标注区域直接打开编辑对话框
- **持久化存储**: 标注数据以 JSON 格式保存到 `data/annotations/` 目录

#### 导航与交互
- **页码导航**: PDF / 多图片模式下显示页码输入框，支持直接跳转指定页面
- **鼠标滚轮滚动**:
  - 默认滚动：30px/步
  - Ctrl + 滚轮：快速滚动 100px/步
  - Shift + 滚轮：精细滚动 10px/步
- **键盘快捷键**: 空格播放暂停、方向键调速、ESC 关闭窗口
- **缩放滑块**: 工具栏提供缩放控制（预留接口）

#### UI 设计
- **深色主题**: 现代化暗色界面设计
- **自定义组件**:
  - ModernButton: 圆角按钮，支持 primary/success/accent/danger 四种配色
  - ModernSlider: 深色主题滑块
  - ProgressBarSlider: 带区域标记的进度条
  - SpeedCurveCanvas: 贝塞尔曲线绘制画布
- **响应式布局**: QSplitter 分割布局，可调整谱面与控制面板比例
- **设置页面**: 文件列表浏览界面应用完整深色主题样式

#### 技术架构
- **异步加载**: QThreadPool + QRunnable 多线程加载文件内容
- **信号槽通信**: Worker 通过自定义信号类传递加载结果
- **虚拟环境**: Python venv 隔离运行环境
- **跨平台兼容**: Windows / Linux / Docker 支持，相对路径配置
