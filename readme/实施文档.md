# 实施文档

## 1. 环境准备

### 1.1 系统要求
- **操作系统**: Windows 10+ / Linux (Ubuntu 20.04+) / Docker
- **Python**: 3.8 或更高版本
- **内存**: 建议 512MB 以上（PDF大文件时需要更多）
- **磁盘**: 项目本身 <5MB，运行时数据根据标注数量增长

### 1.2 安装步骤

#### Windows 环境

```powershell
# 1. 进入项目目录
cd "e:\Projects\TAB Score Viewer"

# 2. 创建虚拟环境（如尚未创建）
python -m venv venv

# 3. 激活虚拟环境
.\venv\Scripts\activate

# 4. 安装依赖（使用清华源加速）
pip install PyQt5 PyMuPDF Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn

# 5. 运行程序
python guitar_tab_viewer.py
```

#### Linux 环境

```bash
# 1. 进入项目目录
cd "/path/to/TAB Score Viewer"

# 2. 创建虚拟环境
python3 -m venv venv

# 3. 激活虚拟环境
source venv/bin/activate

# 4. 安装依赖（使用阿里云源）
pip install PyQt5 PyMuPDF Pillow -i https://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com

# 5. 运行程序
python3 guitar_tab_viewer.py
```

#### Docker 环境

```dockerfile
FROM python:3.11-slim

WORKDIR /app
COPY . .

RUN pip install PyQt5 PyMuPDF Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn

# 需要X11转发或VNC来显示GUI
CMD ["python", "guitar_tab_viewer.py"]
```

```bash
# 构建并运行（需要X11显示支持）
docker build -t tab-viewer .
docker run -it --rm -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix tab-viewer
```

### 1.3 依赖说明

| 包名 | 用途 | 版本 |
|------|------|------|
| PyQt5 | GUI框架，提供窗口、控件、信号槽、绘图等 | >=5.15 |
| PyMuPDF | PDF文件解析与页面渲染为图片 | >=1.23 |
| Pillow | 图片格式解码（PNG/JPG/WEBP等） | >=10.0 |

## 3. 使用指南

### 3.1 基本操作流程

```
启动程序 → 选择文件夹 → 双击文件打开谱面 → 播放/调速/添加标注
```

### 3.2 详细操作说明

#### 打开文件
1. 点击「选择文件夹」按钮
2. 选择存放吉他谱图片/PDF的文件夹
3. 文件列表会自动过滤显示支持的格式
4. 双击任意文件即可打开

#### 播放控制
1. 点击「▶ 播放」或按空格键开始播放
2. 谱面会自动向上滚动
3. 拖动底部进度条可跳转到任意位置
4. 再次点击或按空格暂停

#### 速度调节
1. 拖动速度滑块调整基础速度（值越小越快）
2. 对于变速练习需求，点击「📊 速度曲线」按钮：
   - 在曲线上点击添加控制点
   - 拖动控制点调整位置和速度
   - 勾选「启用速度曲线」生效
   - 可使用预设模板快速应用常用曲线

#### 循环播放
1. 在循环模式下拉框中选择模式
2. 「区域循环(A-B)」模式：
   - 按住 Ctrl 点击进度条设置起点 A
   - 按住 Shift 点击进度条设置终点 B
   - 进度条上会高亮显示 A-B 区域
   - 播放到 B 点后自动回到 A 点

#### 添加标注
1. **方式一**: 在谱面上双击空白处 → 弹出编辑对话框 → 输入文本和样式 → 确定
2. **方式二**: 点击「📝 标注」按钮打开管理器 → 点击「新增」→ 编辑 → 确定
3. **编辑已有标注**: 双击谱面上的标注文字，或在管理器中选中后点「编辑」

#### 页码导航（多图/PDF模式）
1. 底部左侧出现页码输入框（格式: `[输入框] / 总页数`）
2. 直接输入页码数字后按回车跳转
3. 滚动时页码输入框自动同步当前可见页

#### 鼠标滚轮
- 在谱面区域使用鼠标滚轮上下翻页
- Ctrl + 滚轮：快速翻页
- Shift + 滚轮：精细移动

### 3.3 标注数据说明

标注以 JSON 格式存储在 `data/annotations/` 目录下：

```json
[
  {
    "id": "ann_a1b2c3d4",
    "x": 0.5,
    "y": 0.3,
    "text": "这里用泛音演奏",
    "color": "#F97316",
    "font_size": 14,
    "font_family": "Microsoft YaHei",
    "is_bold": false,
    "background_color": "#00000080"
  }
]
```

字段说明:
- `id`: 唯一标识符
- `x`, `y`: 相对坐标 (0.0-1.0)，(0,0)=左上角, (1,1)=右下角
- `text`: 标注文本内容
- `color`: 文字颜色（十六进制）
- `font_size`: 字体大小（像素）
- `is_bold`: 是否粗体
- `background_color`: 背景色（支持透明度）

## 4. 常见问题

### Q1: 程序启动报错 `ModuleNotFoundError`
**原因**: 依赖未安装或虚拟环境未激活
**解决**:
```bash
# 确认已激活虚拟环境（命令行前缀应有 (venv)）
.\venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux

# 重新安装依赖
pip install PyQt5 PyMuPDF Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: PDF 文件打不开
**原因**: PyMuPDF 未正确安装或 PDF 文件损坏
**解决**:
```bash
pip uninstall PyMuPDF
pip install PyMuPDF -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3: 中文显示乱码
**原因**: 终端编码问题
**解决**: Windows 下设置环境变量:
```powershell
$env:PYTHONIOENCODING="utf-8"
python guitar_tab_viewer.py
```

### Q4: WEBP 格式无法加载
**原因**: Pillow 版本过低不支持 WEBP
**解决**:
```bash
pip install Pillow>=10.0 --upgrade
```

### Q5: 大 PDF 文件加载慢
**原因**: PDF 渲染为图片是 CPU 密集型操作
**解决**: 正常现象，异步加载不会阻塞界面。首次加载后会缓存。

## 5. 性能参数参考

| 参数 | 默认值 | 说明 | 调整效果 |
|------|--------|------|----------|
| base_speed | 50ms | 基础播放速度 | 值越小越快，范围1-500 |
| scroll_step | 自动计算 | 每帧滚动像素 | 由 speed 自动映射 |
| undo_depth | 50 | 撤销最大步数 | 增大占用更多内存 |
| wheel_step | 30px | 滚轮默认步长 | 增大滚动更快 |

## 6. 备份与迁移

### 备份重要数据
只需备份以下目录/文件：
- `config/settings.json` - 用户设置
- `data/annotations/` - 所有标注数据

### 迁移到新机器
1. 复制整个项目文件夹到新机器
2. 安装 Python 3.8+
3. 创建虚拟环境并安装依赖
4. 将备份的 `config/` 和 `data/` 目录覆盖回去
5. 运行程序
